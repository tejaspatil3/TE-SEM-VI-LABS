
1. What is Cloud Computing with example?
Cloud computing is the delivery of computing services like storage, servers, databases, networking, software over the internet. Example: Google Drive lets you store files online and access them from anywhere.

2. Who is the father of cloud computing?
Dr. Joseph Carl Robnett Licklider is known as the father of cloud computing.

3. List down the basic characteristics of cloud computing?
- On-demand self-service
- Broad network access
- Resource pooling
- Rapid elasticity
- Measured service

4. What do you mean by Vertical and Horizontal scaling in Cloud Computing?
- Vertical scaling: Adding more power (CPU, RAM) to an existing machine.
- Horizontal scaling: Adding more machines to handle load.

9. Explain the difference between cloud and traditional datacenters?
Cloud is flexible, pay-as-you-go, and accessed over the internet. Traditional datacenters require physical setup, high upfront cost, and manual management.

19. What is Virtualization?
Virtualization is creating a virtual version of something like server, storage device, or network resource.

20. What is an EC2 instance?
An EC2 instance is a virtual server in Amazon's Elastic Compute Cloud (EC2) for running applications on AWS.

21. What is AMI in EC2?
AMI (Amazon Machine Image) is a template that contains software configuration (OS, app server, apps) to launch EC2 instances.

22. How to launch an on-demand EC2 instance in AWS?
- Login to AWS Console
- Open EC2 dashboard
- Click "Launch Instance"
- Choose AMI, instance type, configure settings
- Launch

23. What is Google App Engine?
Google App Engine is a cloud platform for developing and hosting web apps in Google-managed data centers.

24. What are advantages of Google App Engine?
- No server management
- Auto-scaling
- Built-in services like NoSQL datastores
- Secure and reliable

25. What are the steps to install and configure Google App Engine?
- Install Google Cloud SDK
- Initialize SDK with `gcloud init`
- Write your app code
- Deploy using `gcloud app deploy`

32. What is Lightning Platform?

36. What is hypervisor in Cloud Computing?
A hypervisor is software that creates and runs virtual machines (VMs). It sits between hardware and operating systems.

37. Explain Load Balancing in Cloud Computing.
Load balancing distributes incoming network traffic across multiple servers to ensure no single server is overwhelmed.
