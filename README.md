**TE SEM-VI Computer Engineering SPPU 2019 Pattern**
This repository contains laboratory assignments, source codes, and syllabus resources for Semester 6 of the Third Year (TE) Computer Engineering course at Savitribai Phule Pune University (SPPU), following the 2019 Pattern.

Repository Overview
The content covers practical assignments for Semester 6, specifically focusing on Laboratory Practice 2 (LP-2), Data Science and Big Data Analytics (DSBDA), and Web Technology (WT).

Directory Structure
**1. DSBDA (Data Science and Big Data Analytics)**
This directory contains Python scripts and Jupyter Notebooks for data analysis and visualization.

Assignments:

Implementation of various data processing tasks (practi-2.ipynb to practi-11.ipynb).

MapReduce implementation in Python.

Data visualization and analysis assignments.

Resources:

Group A and Group B specific assignments.

Input/Output datasets and logs.

**2. LP-2 (Laboratory Practice 2)**
This section is divided into two main domains: Artificial Intelligence (AI) and Cloud Computing (CC).

A. LP-2 AI (Artificial Intelligence)
Algorithms & Codes:

Implementation of standard AI algorithms (BFS, DFS, A*, etc.).

Mini-projects and separated code modules.

Resources:

Lab Manuals and Notes.

Input/Output references.

B. LP-2 CC (Cloud Computing)
Assignments:

Cloud simulation and setup assignments.

Assignment 3 & 4 implementations.

Mini-projects.

Resources:

Lab Manuals (2024-2025).

**3. WT (Web Technology)**
This directory covers full-stack web development assignments.

Assignments:

Assignment 2-5: Core web technology tasks.

Assignment 6: Java Web Application structure (src/main/webapp).

Assignment 7: Advanced assignment implementation.

Assignment 9: Angular registration master project.

Resources:

Web Technology Laboratory Manual.

Output screenshots and logs.

Prerequisites and Tools
To execute the codes in this repository, the following tools are recommended:

Anaconda / Jupyter Notebook: For DSBDA Python assignments.

Java Development Kit (JDK): For AI and Web Technology assignments.

IDE: VS Code, IntelliJ IDEA, or Eclipse.

Web Frameworks: Angular (for WT Assignment 9), Apache Tomcat (for Java Web Apps).

Cloud Simulators: Salesforce or Google App Engine (for CC).

Usage
Clone the repository to your local machine.

Navigate to the specific subject folder.

Follow the instructions in the respective lab manuals for execution.

Disclaimer
These codes are provided for educational and reference purposes to assist with understanding the SPPU 2019 Pattern syllabus. Students are encouraged to implement and test the logic independently.
